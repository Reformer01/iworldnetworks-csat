# **App Name**: I-World Connect

## Core Features:

- Customer Feedback Portal: A user-friendly public portal allowing customers to submit feedback on various categories including reliability, support, testimonials, installation, and billing.
- Admin Telemetry Dashboard: A secure administrative hub to view and analyze key operational metrics and performance indicators across network health, installation efficiency, billing reconciliation, and support desk performance.
- Dynamic Reporting Module: Admins can generate comprehensive reports based on aggregated customer feedback and operational data, highlighting trends and areas for improvement.
- AI-Powered Sentiment Insights Tool: Utilize generative AI to analyze qualitative feedback (e.g., from testimonials and comments) and provide actionable sentiment insights for administrative decision-making.
- Secure PIN Authentication: Implement a dedicated PIN-based login system for administrator access to the telemetry dashboards, ensuring secure entry to sensitive operational data.
- Interactive Customer Engagement Metrics: Display real-time customer satisfaction (CSAT) index, feedback volume, and testimonial curation for marketing and service enhancement.
- Field Technician Performance Tracking: Monitor and display metrics related to installation efficiency, completion rates, and average resolution times for individual field technicians.

## Style Guidelines:

- The design embodies a clinical, high-tech aesthetic, emphasizing clarity, precision, and reliability. This is best achieved with a predominantly cool, minimalist palette. The overall scheme is light.
- Primary interactive color: A vibrant yet deep corporate blue (#0058BE), chosen for its strong association with technology and trust, providing clarity for calls to action and key data points. HSL(212, 100%, 37%).
- Background color: A serene, heavily desaturated light grey (#F8F9FA) creates a clean canvas, echoing the crispness of modern data centers. HSL(210, 10%, 98%).
- Accent color: A deep, muted cyan-gray (#2E4B51) for subtle visual differentiation and added depth, providing good contrast against the primary blue and light background. HSL(195, 27%, 25%).
- Headlines and body text: 'Outfit' (sans-serif), lending a modern, structured, and highly legible appearance across all content, from bold titles to extended paragraphs.
- Labels, data, and technical snippets: 'JetBrains Mono' (monospace sans-serif), used for its crisp, developer-friendly aesthetic that conveys precision and order in numerical data and categorical tags.
- Utilize 'Material Symbols Outlined' for all iconography, configured for optimal readability and a unified aesthetic (FILL 0, Wght 400, GRAD 0, opsz 24). Specific icons like dashboard, support_agent, and reviews reinforce functionality.
- The layout strategy incorporates asymmetric grids and 'bento-box' inspired modules, providing a dynamic yet organized presentation of information with generously rounded corners (up to 2.5rem) and subtle 'whisper-shadow' for depth.
- Subtle, smooth animations (e.g., hover effects with cubic-bezier transitions for a 'spring-like' feel, grayscale-to-color image reveals on hover, loading shimmers, and gentle parallax on elements) enhance user interaction without distraction.
